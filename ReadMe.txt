The following libraries where used and are required for the program to run:
HaLex 			(http://hackage.haskell.org/package/HaLeX-1.2.6)
unordered-containers 	(https://hackage.haskell.org/package/unordered-containers-0.2.13.0)
sort			(https://hackage.haskell.org/package/sort-1.0.0.0) 

They can be installed by running:
cabal install Halex unordered-containers sort
